## Pré-requis

- Docker

## Installation

```bash
curl -O https://raw.githubusercontent.com/Badyssblt/wra602d-frontend/main/compose.hub.yml
docker compose -f compose.hub.yml up -d
```

- App → http://localhost:8080
- Mails → http://localhost:8025
